# Kill the VC - Installation Instructions

Thank you for purchasing Kill the VC from ThePegasusAI!

## Installation

1. Drag the "kill-the-vc.app" icon to the Applications folder to install the game.
2. Open the application from your Applications folder.
3. On first run, you'll be asked to accept the license agreement.
4. Enter your license information when prompted.

## System Requirements

- macOS 10.14 or later
- 4GB RAM minimum (8GB recommended)
- 500MB free disk space
- Webcam required for hand gesture controls
